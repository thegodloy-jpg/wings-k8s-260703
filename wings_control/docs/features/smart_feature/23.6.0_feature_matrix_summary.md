## 后续白名单对齐关注点

- Qwen3.6 在最新矩阵中同时覆盖 Ascend 910C BF16、Ascend 910B3 INT8、Ascend 910C INT8，均为 SmartDecoding。
- GLM-5.2-w8a8 在最新矩阵中是 Ascend 910C SmartDecoding，不包含 SmartKVCache 或 SmartKVSparse。
- DeepSeek-V4-Flash 在 Ascend 侧最新矩阵只标 SmartKVCache；在 NVIDIA 侧最新矩阵只标 SmartDecoding。
- GLM-5.1 在 Ascend/NVIDIA 侧均只标 SmartKVSparse。
- 白名单更新应按三张独立表分别对齐：spec、offload、sparse；不要按模型默认配置或 parser 支持自动全量加入。
